# Meu Primeiro Projeto PHP: Simulador de Banco Básico

**Descrição:**

Este projeto é um simulador de banco simples desenvolvido em PHP para fins de aprendizado. A aplicação permite ao usuário realizar operações básicas como depósitos, saques e consulta de saldo.

**Funcionalidades:**

**Depósito:** Permite adicionar um valor ao saldo atual.

**Saldo:** Exibe o valor total do saldo.

**Saque:** Permite retirar um valor do saldo, verificando se há saldo suficiente.

**Sair:** Encerra a aplicação.

**Tecnologias Utilizadas:**
PHP
